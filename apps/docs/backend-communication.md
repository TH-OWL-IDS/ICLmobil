# Backend Communication

The app uses REST services with the help of the Axios plugin to communicate with the backend. The services are logically structured according to the API routes they contain.

**Service Files:**

Sharing-related functions  
`/src/services/assetService.js`

Bookings and rides  
`/src/services/bookingService.js`

Messages and push notifications  
`/src/services/messageService.js`

News-related functions  
`/src/services/newsService.js`

Points of interest functions  
`/src/services/poiService.js`

Settings-related functions  
`/src/services/preferencesService.js`

App-specific functions  
`/src/services/systemService.js`

User-specific functions  
`/src/services/userService.js`
