SMS Verifikation

Um sicherzustellen, dass eine Telefonnummer zu der Person gehört, die sie in die App einträgt, wird eine SMS mit einem Code verschickt. Das Senden von SMS geschieht im Hintergrund, nachdem eine Handynummer eines Benutzers geändert wurde. Außerdem lässt sich in den Benutzer-Einstellungen oben rechts ein Knopf für den manuellen Versand finden.

Twilio Guthaben: {{ balance.0 }} {{ balance.1 }}

Alternativ kann unten manuell ein SMS-Versand ausgelöst werden.
