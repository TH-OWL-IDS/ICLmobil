# Support texts

Support texts appear in the app under the gear icon -> App information -> Help with the exception of entries that
have a special value in the field `Name of entry`:

- `gtc`: Terms and Conditions
- `dataprotection`: Data Protection
- `imprint`: Imprint
- `about`: About
- `account-deletion`: How to delete my user account?

## Special fields in support texts

The `Sort order` controls in which order the entries of a category (like FAQ) appear in the mobile app.
Each category also has an order which controls the order of the categories in the mobile app.

The list view shows it like this `10 -> 12` which means the category has a sort order of 10 and the support text has 12.
